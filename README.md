
# Classical Music Themed Escape Room

Welcome to the Classical Music Themed Escape Room! This is an interactive web-based escape room game where each room is inspired by elements of classical music. Solve puzzles, explore different rooms, and test your music knowledge!

Click here to see the live site: https://hannahro15.github.io/classical-music-themed-escape-room/

## Features
- Multiple themed rooms, each with unique puzzles
- Classical music concepts and trivia
- Engaging visuals (all room images generated by Google Gemini)
- Built with React and Vite for a fast, modern experience

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation
1. Clone this repository:
	```bash
	git clone https://github.com/your-username/classical-music-themed-escape-room.git
	cd classical-music-themed-escape-room
	```
2. Install dependencies:
	```bash
	npm install
	# or
	yarn install
	```

### Running the App
Start the development server:
```bash
npm run dev
# or
yarn dev
```
Then open [http://localhost:5173](http://localhost:5173) in your browser.

## Project Structure

- `music-escape-room/` - Main app source code
  - `src/` - React components, styles, and assets
  - `public/` - Static files
  - `rooms/` - Individual room and puzzle components
